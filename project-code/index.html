What is Terraform?

Terraform is an open-source Infrastructure as Code (IaC) tool developed by HashiCorp. It allows users to define and provision data center infrastructure using a high-level configuration language known as HCL (HashiCorp Configuration Language).

Key Features

  • Declarative configuration language (HCL)
  • Supports multiple providers (AWS, Azure, GCP, etc.)
  • Execution plans and change automation
  • Resource graph for dependency resolution
  • Modular and reusable configurations

Basic Workflow

  1. Write: Define infrastructure in `.tf` files using HCL
  2. Initialize: Run terraform init to prepare your environment
  3. Plan: Use terraform plan to preview changes
  4. Apply: Execute terraform apply to provision resources
  5. Destroy: Run terraform destroy to clean up

Use Cases

  • Provisioning cloud infrastructure
  • Setting up multi-cloud environments
  • Managing Kubernetes clusters
  • Automating infrastructure deployments
